Kernel Trick¶

Un "kernel" est une "fonction noyau" (ou simplement "noyau") qui permet de reprsenter des donnes d'un espace donn dans un autre espace, souvent de dimension plus grande.

Dans notre implmentation, pour pouvoir utiliser diffrents types de fonction noyau, nous dfinissons une classe abstraite :

In [8]:
class Kernel():
    """ Classe pour reprsenter des fonctions noyau
    """
    def __init__(self, dim_in, dim_out):
        """ Constructeur de Kernel
            Argument:
                - dim_in : dimension de l'espace de dpart (entre du noyau)
                - dim_out: dimension de l'espace de d'arrive (sortie du noyau)
        """
        self.input_dim = dim_in
        self.output_dim = dim_out
        
    def get_input_dim(self):
        """ rend la dimension de l'espace de dpart
        """
        return self.input_dim

    def get_output_dim(self):
        """ rend la dimension de l'espace d'arrive
        """
        return self.output_dim
    
    def transform(self, V):
        """ ndarray -> ndarray
            fonction pour transformer V dans le nouvel espace de reprsentation
        """        
        raise NotImplementedError("Please Implement this method")

L'intrt de dfinir une classe mre Kernel qui servira dfinir des noyaux et que l'on pourra maintenant dfinir des algorithmes gnriques qui pourront prendre des noyaux quelconques en argument.

 Un premier noyau¶

Le premier noyau, trs simple, que l'on dfinit permet de projeter (manuellement) des donnes 2D dans un espace de plus grande dimension.

Voici un exemple de projection qui transforme le vecteur $(x_1,x_2)$ en vecteur $(x_1,x_2,1)$

In [9]:
class KernelBias(Kernel):
    """ Classe pour un noyau simple 2D -> 3D
    """
    def transform(self, V):
        """ ndarray de dim 2 -> ndarray de dim 3            
            rajoute une 3e dimension au vecteur donn
        """
        V_proj = np.asarray([V[0],V[1],1])
        return V_proj

Chargement du fichier "base_2D.csv"¶

Les datasets classiques qui sont disponibles sur le web sont gnralement fournis au format CSV (Comma Separated Values) prsent dans le cours 2: chaque exemple est donn sur une ligne, et sur cette ligne les valeurs de description sont spares par une virgule. Le plus souvent, le label de l'exemple est aussi donn sur cette ligne, tout la fin.

Pour lire un fichier CSV et pouvoir travailler dessus, nous utilisons la commande pd.read_csv() de la librairie pandas. Cette fonction prend en argument une chane de caractres qui donDataFramene le nom du fichier ouvrir (avec le chemin d'accs s'il n'est pas dans le rpertoire courant) et elle rend un DataFrame.

Pour tester, utiliser le fichier base-2D.csv qui se trouve dans le rpertoire data. On peut noter que la premire du fichier contient le nom des colonnes qui seront donnes au dataframe cr. Si ces noms sont absent (fichier csv brut), consulter l'aide sur cette fonction pour voir comment l'utiliser.
